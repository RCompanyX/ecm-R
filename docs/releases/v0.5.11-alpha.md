# What's New

## Experimental In-Game Movie Muting

### New Overlay Toggle

- ECM-R now exposes an `Experimental` overlay menu with an `In-Game Movie Muting` toggle.
- The toggle updates the runtime behavior immediately and persists the change back to the INI file.

### Disabled by Default

- This feature remains experimental in this release.
- The new `[experimental]` `ingame_movie_muting` setting defaults to `false`.
- While the setting stays disabled, ECM-R keeps the legacy package-driven pause and resume behavior.

### Comic-Style Movie Coverage

- When enabled, ECM-R treats NFSU2's documented `IG_PlayMovie.fng` package as an in-game movie mute trigger.
- This covers comic-style career movie sequences that play before or after races through the game's `IG_PlayMovie.fng` path.
- The experimental mode also reconciles pause state against the mute packages that are actually loaded so music stays paused for the full lifetime of the movie package.

## Configuration Changes

### Experimental Section

- The movie-muting option now lives under a dedicated `[experimental]` section as `ingame_movie_muting = false`.
- Existing temporary `experimental_ingame_movie_muting` entries migrate automatically to the new setting name and section.

## Documentation Updates

- Updated `README.md` to describe the experimental overlay menu entry and the new INI setting.
- Updated `CONFIGURATION.MD` to document the `[experimental]` section and the default `false` value.
- Updated `docs/application-context.md` to record that the movie-muting path is still experimental.
- Updated `CHANGELOG.md` for `v0.5.11-alpha`.

## Validation

- Built successfully in `Release | Win-x86`.
- Verified touched source files for editor-reported errors.
- Manual gameplay validation confirmed that enabling the option pauses music during a comic-style race cinematic and resumes playback on race entry.

## Notes

- This release keeps the feature experimental on purpose.
- If `stop_music_on_loading_screens` is enabled, loading screens can still stop the current track after the movie transition.
