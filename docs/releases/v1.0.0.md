# What's New

## Stable Release

- ECM-R is now published as `v1.0.0`, the first stable release of this fork for **Need for Speed: Underground 2**.
- This release keeps the established runtime layout and deployment flow:
  - `scripts/ecm-r.x86.asi`
  - `scripts/ecm-r.x86.ini`
  - `scripts/bass.dll`
  - `scripts/Music/`
- No gameplay-state, playlist, overlay, or audio-flow behavior changes are introduced by this release cut.

## Runtime and Configuration

- The runtime version marker written to generated or refreshed `ecm-r.x86.ini` files is now `v1.0.0`.
- Existing configuration keys and runtime filenames remain unchanged for compatibility with current installations.
- The official `bass.dll` runtime requirement is unchanged; users must continue downloading it from the official BASS website.

## Packaging and Support Readiness

- Release documentation has been aligned for the stable `v1.0.0` package.
- The bug report issue template now includes `v1.0.0` so incoming reports can target the stable release directly.
- Changelog release references now point to the ECM-R repository and include the missing `v0.5.6-alpha` tag link.

## Validation Notes

- The documented release build flow remains `generate.bat` on Windows, followed by `Release | Win-x86` in the generated Visual Studio solution.
- This release prep does not change the expected BASS runtime deployment or the `scripts` folder layout.
